## Что нужно сделать СРОЧНО:

### 1. Ротация ключей в Supabase:

1. Откройте Supabase Dashboard
2. Перейдите в **Settings** → **API**
3. Сгенерируйте новые ключи:
   - **ANON_KEY** - новый ключ
   - **SERVICE_ROLE_KEY** - новый ключ (если используется)
4. Обновите переменные окружения в приложении

### 2. Изменение паролей:

- **SSH пароль сервера** - измените немедленно
- **Пароль Supabase** - измените в настройках
- **Пароль PostgreSQL** - измените если возможно

### 3. Запуск приложения с переменными окружения:

```bash
# Для разработки
flutter run --dart-define=SUPABASE_URL=http://your-server:8000 --dart-define=SUPABASE_ANON_KEY=your_new_key

# Или установите переменные окружения системы
export SUPABASE_URL=http://your-server:8000
export SUPABASE_ANON_KEY=your_new_key
flutter run
```

### 4. Для продакшена:

Используйте переменные окружения вашей платформы развертывания.
