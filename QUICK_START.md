# Быстрый старт после исправления безопасности

## ⚠️ ВАЖНО: Приложение теперь требует переменные окружения!

После исправления безопасности все секреты удалены из кода. Для запуска приложения нужно установить переменные окружения.

## Способ 1: Через командную строку (рекомендуется)

```bash
flutter run --dart-define=SUPABASE_URL=http://your-server:8000 --dart-define=SUPABASE_ANON_KEY=your_anon_key
```

## Способ 2: Через переменные окружения Windows

```powershell
# Установить переменные окружения
$env:SUPABASE_URL="http://your-server:8000"
$env:SUPABASE_ANON_KEY="your_anon_key"

# Запустить приложение
flutter run
```

## Способ 3: Создать .env файл (для локальной разработки)

1. Скопируйте `env.example` в `.env`
2. Заполните реальными значениями
3. Используйте пакет `flutter_dotenv` для загрузки (см. SETUP_ENV.md)

## Где взять значения:

- **SUPABASE_URL**: URL вашего Supabase сервера (например: `http://176.124.208.227:8000`)
- **SUPABASE_ANON_KEY**: Anon Key из Supabase Dashboard → Settings → API

## После ротации ключей:

Если вы ротировали ключи после обнаружения проблемы безопасности:
1. Используйте новые ключи в переменных окружения
2. Старые ключи больше не работают
